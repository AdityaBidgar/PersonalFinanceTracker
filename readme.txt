# Personal Finance Tracker

A simple and intuitive Personal Finance Tracker built using **Python**, **Streamlit**, and **Supabase**. This web app allows users to manage their income and expenses, visualize their spending habits, and stay on top of their financial goalsâ€”all in one place.

## Features

- Add income and expense entries
- Categorize transactions (e.g., Food, Rent, Salary, etc.)
- Visualize spending with interactive charts
- Monthly summary and balance overview
- Secure cloud storage with **Supabase** backend
- Intuitive UI with **Streamlit**

## Tech Stack

- **Frontend & App Framework**: [Streamlit](https://streamlit.io/)
- **Backend & Database**: [Supabase](https://supabase.io/)
- **Language**: Python
